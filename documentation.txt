5.Description     : The art and science of hiding information by embedding messages within other, 
                    seemingly harmless messages. Bits of unused data are replaced by bits of valuable 
                    information using LSB mechanism. Sender and receiver will have individual key / secret 
                    based on which they will be able to extract the actual data from the image. 
                    This project also gives basic level understanding of image processing methodologies.

6.Sample input and outputs:

   Key Features:
   - Encodes text files into 24-bit BMP images
   - Decodes hidden messages from stego images
   - Supports multiple file formats (.txt, .c, .sh, .h)
   - Verifies image capacity before encoding
   - Uses magic string "#*" for stego image identification
   - Maintains original image dimensions and quality

   Technical Implementation:
   - Each secret byte (8 bits) is distributed across 8 image bytes (1 bit per byte)
   - BMP header (54 bytes) is preserved unchanged
   - Encodes file metadata (extension, size) along with content
